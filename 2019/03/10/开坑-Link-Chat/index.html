开坑 Link Chat 2019-03-10

由头

提前准备一个毕业设计，打算做一款随机匹配聊天、匿名的 Web Chat Application，于是就有了这么一个想法

前端打算用 Vue + Vuex + Nuxt + iView + Socket.IO-Client 实现

选用 iView 是因为之前没什么机会用它，ElementUI用的有点腻了（#Update 20190318 iView 使用下来没有 ElementUI 舒服，以后不用了 =3=）

后端打算用 Egg + Redis + Socket.IO-Server 实现

后端不采用 MongoDB 或 Mysql 是因为暂时没有保存聊天记录的需求，Redis足以Handle目前的场景
